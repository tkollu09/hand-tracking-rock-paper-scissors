# hand-tracking-rock-paper-scissors
A python rock paper scissors game made with mediapipe

## Usage

1. **Clone the repository**
```bash
   git clone https://github.com/<your-username>/hand-tracking-rock-paper-scissors.git
```
2. **Install the required packages**  
```bash
  pip install -r requirements.txt
```
3. **Run the game**
```bash
  python game.py
