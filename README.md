# hand-tracking-rock-paper-scissors
A python rock paper scissors game made with mediapipe

## Use

1. **Clone the repository**
git clone hand-tracking-rock-paper-scissors
2. **Install the required packages**  
Make sure you have Python 3.x installed, then install all dependencies from `requirements.txt`:
pip install -r requirements.txt
3. **Run the game**
python game.py
